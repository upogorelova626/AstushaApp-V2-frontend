# AstushaApp

**AstushaApp** — веб-приложение для управления командами, проектами и задачами в экосистеме **Astusha**.

Приложение помогает организовать рабочий процесс: создавать команды, вести проекты, назначать задачи участникам, прикреплять файлы, добавлять ссылки на репозитории и обсуждать задачи в комментариях.

AstushaApp является частью экосистемы Astusha и использует **Astusha ID** как единый сервис идентификации пользователей. Авторизация, регистрация, восстановление доступа, профиль аккаунта и безопасность управляются через Astusha ID, а AstushaApp отвечает за прикладную логику: команды, проекты, задачи, комментарии и файлы.

Проект разрабатывается как fullstack pet-project с frontend на Angular и backend на NestJS.

## Экосистема Astusha

В экосистеме Astusha пользователь имеет единый аккаунт, который используется во всех сервисах.

**Astusha ID** отвечает за:

* регистрацию и авторизацию;
* хранение единого профиля пользователя;
* управление email, паролем и безопасностью;
* выдачу и обновление JWT-токенов;
* завершение сессии пользователя.

**AstushaApp** отвечает за:

* команды;
* проекты;
* задачи;
* участников;
* комментарии;
* файлы;
* ссылки на репозитории;
* прикладные настройки рабочего процесса.

При первом входе пользователя через Astusha ID backend AstushaApp автоматически создаёт локальную запись пользователя в своей базе данных с тем же `id`, что и в Astusha ID. Это позволяет использовать единый идентификатор пользователя во всей экосистеме.

## Возможности

В приложении реализованы следующие возможности:

* вход через Astusha ID;
* автоматическое создание локального пользователя AstushaApp после входа через Astusha ID;
* просмотр профиля пользователя;
* создание команд;
* просмотр команд и участников;
* управление участниками команды;
* создание проектов;
* просмотр доски проекта;
* перемещение задачи между стадиями;
* настройка проекта;
* управление участниками проекта;
* роли в проекте;
* создание задач;
* редактирование задач;
* удаление задач;
* назначение исполнителя задачи;
* типы и приоритеты задач;
* story points и дедлайны;
* workflow-стадии задач;
* прикрепление файлов к задаче;
* удаление файлов задачи;
* комментарии к задачам;
* ссылки на репозитории проекта;
* страница одной задачи;
* страница одного проекта;
* адаптивный интерфейс.

> Регистрация, восстановление доступа, смена пароля, редактирование аккаунта, аватар и безопасность пользователя управляются через Astusha ID.
